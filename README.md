## About
- 주로 많이 키우는 반려동물(ex:강아지,고양이)이 아닌 조금은 특별한 반려동물을 위한 커뮤니티 제공
- 반려동물종류에 따른 전용 게시판이 있어 같은동물을 키우는 사람끼리 정보 & 일상 공유 가능
- 마이페이지에 반려동물의 이름과 출생일을 기록 및 저장하여 조회가능.
